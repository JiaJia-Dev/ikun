# ikun

说来奇怪哈，来这边说点小心事，说些想法。明明是代码的程序员的平台哈。还借用了坤哥的名气。
感觉许多话还是很刺痛人的，你会不会包容你的孩子，你的孩子又会不会包容你的。我想你不会说孩子包容你，只会说自己包容孩子。孩子也不会说包容了你。
包容这个词语吧，天生就自带了上级对待下级的味道。大概会说自己包容的人的心理历程是你错了（尽管他们本人自己都没有发觉这个过程），然后紧接着跟孩子说没事（一个孩子把碗摔了），跟孩子说没事，是因为你本身默认了这是一件错事，（这里我还想提一嘴，不是说错了才可以小心一点，不犯错。对错和怎么做事情，做不做事情其实是解绑的，但是我们总爱绑在一起，我的错，所以我要改，你的错，所以你要改）对错的来历就是你认为哪一个更好，他可能是损害自己利益的，可能是保护自己的。（这里就当少挣点钱，多麻烦事是损害利益的，不扯心理上的了）
没事的这个词也隐含了一个倾向，孩子一开始摔了碗只是被吓到了，知道你跟他说没事的，错的概念和自己是下位的地位也传给了孩子。我再强调一下，并不是认识不到错误就不会改变，孩子不会喜欢惊吓。认识到什么对的错的和一个人做不做事怎么做事，本来就不是强绑定的。
我一个朋友跟我分享了视频，一个小孩子好像是和小区大爷有了点什么矛盾。然后哥哥教育着孩子，要为人民服务，要吃苦，要懂得隐忍，伺机而动。当然这些是建立在灌输上的，是哥哥告诉孩子你要为人民服务（我不是在否定着一个你艰辛维护的信念，孩子看到了这个人不高兴，那个人受苦了，会有一天来向你问，为什么他们那么辛苦啊，这才是一个孩子自己长出来的。小孩哭了，听哥哥说了这么多，然后其实哥哥还是蛮欣慰的，谁来看都会感觉是皆大欢喜啦。
我想，如果孩子是学到了东西，如果孩子长出了不要让他们那么辛苦的心，他只会继续追问下去，那我要怎么做的更好的，又或者是一副恍然大悟的样子，自己学到了东西哦，更知道自己要怎么做了。（如果你真的在做事情，其实你不会有自己爱国是多么高尚的行为，你只是在爱着，在想着怎么做）
真的在学习的孩子是不会因为一个哥哥说了这些开始愧疚的哭了起来的，更像是一个很有默契的表演，两个演员都不知道自己在干什么。（该被刺痛的还是会被刺痛的，如果你只是拿着爱国的这个名号再做点别的什么，如果你没有那么大的勇气看看自己在干嘛，看看世界在干嘛）
